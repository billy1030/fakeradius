<task type="auto">
  <name>Task 2: Implement RADIUS server core (UDP listener, packet codec, Message-Authenticator)</name>
  <files>cmd/server/main.go, cmd/server/radius.go</files>
  <read_first>None — new files</read_first>
  <acceptance_criteria>Server binary builds with -secret flag, listens on UDP 1812, handles packets</acceptance_criteria>
  <action>
    Create the RADIUS server foundation:

    **cmd/server/main.go** (entry point):
    - Define a Secret flag (-secret string) using pflag for the shared secret
    - Start UDP server listening on ":1812" using net.ListenPacket("udp", ":1812")
    - Read Access-Request packets from the UDP socket
    - Parse each packet using radius.ParsePacket (from radixdlt/radius)
    - Validate Message-Authenticator if present using radius.ValidateMessageAuthenticator
    - Dispatch to handler.Handler for processing
    - Send response packets back to client IP:port
    - Log all requests and responses to stdout

    **cmd/server/radius.go** (RADIUS utilities):
    - Define Packet struct wrapping radius.Packet with helper methods
    - Implement BuildAccessAccept(packet *radius.Packet, secret string, replyMessage string) []byte
    - Implement BuildAccessReject(packet *radius.Packet, secret string, replyMessage string) []byte
    - Implement ValidateMessageAuthenticator(packet *radius.Packet, secret string) bool
    - Use RFC 2869 Message-Authenticator attribute (Type 80) for validation
    - Include Reply-Message attribute (Type 18) in responses

    Message-Authenticator calculation: HMAC-MD5 of entire RADIUS packet with Message-Authenticator field zeroed, using shared secret as key.
  </action>
  <verify>
    <automated>go build -o fakeradius-server ./cmd/server && ./fakeradius-server -h 2>&1 | grep -q secret</automated>
  </verify>
  <done>Server binary builds with -secret flag, listens on UDP 1812, handles packets</done>
